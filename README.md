Interactive Excel Dashboard Analyzing E-commerce Sales Data.

#Overview This Project Analyzes E-commerce Sales Data Using Excel to Help Understand Sales trend and Top-performing Products.

#Tools Used

   ♦ Excel
   
   ♦ Power Query
   
   ♦ Pivot Tables
  
#Key Insights

   ♦ 13-week trend shows fluctuating but overall stable growth, with peaks around weeks 6, 9, and 11.
   
   ♦ February recorded the highest sales, followed by March, then January.
   
   ♦ Female customers dominate sales (52%).
   
   ♦ App & Website are the strongest channels across all genders.
   
   ♦ Most orders contain 2–3 items.
   
   ♦ Majority of orders delivered within 1–3 days.
   
   ♦ Top-selling categories are T-shirts, Jeans, Sneakers.
